# 1. Entendimiento del problema

Esta carpeta concentra todo lo relacionado con el entendimiento del problema a lo largo del proyecto. El entendimiento se construye de forma incremental: parte del análisis del primer dataset disponible y se amplía a medida que se incorporan nuevas fuentes de datos.
